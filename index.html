My Projects

Super Spy Cam

I built a personal project, Super Spy Cam. It’s a motion-triggered surveillance system written in Python that uses a webcam to monitor your doorway, record video once it detects movement, and sends real-time alerts (with video attachments) to a Discord channel using the official Discord HTTP API. It also includes a lightweight HTTP server to browse the captured footage from your local browser.
I built this project after a package was stolen from outside my apartment. My plan is to mount a webcam behind the peephole of my front door and connect it to a spare computer (which will run this app).

Technologies Used:

  • Language: Python 3
  • Camera Input: cv2.VideoCapture() from OpenCV
  • Video Encoding: OpenCV cv2.VideoWriter
  • Discord Integration: Python requests library using Discord Bot Token & Channel API
  • Web Server: Python http.server module via HTTPServer and BaseHTTPRequestHandler
  • Frontend: HTML, JavaScript (fetch API), JSON
  • Filesystem: Python os, datetime, and json modules for file naming, lookup, and timestamp formatting

Storage Assignment

I built a small interactive webpage named Storage Assignment, as part of my first-semester Web Programming and Design course at Humber College. The goal of this project was to demonstrate my understanding of DOM manipulation and browser storage using JavaScript.
The app allows users to enter their name, save it using localStorage, and receive a personalized greeting that will stay on the page after the page is refreshed. Users can also clear the saved name with a click. As a bonus feature, the project displays a friendly message based on the current day of the week and time of day using JavaScript’s Date object.
