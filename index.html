My Projects

Super Spy Cam

I built a personal project, Super Spy Cam. Itâ€™s a motion-triggered surveillance system written in Python that uses a webcam to monitor your doorway, record video once it detects movement, and sends real-time alerts (with video attachments) to a Discord channel using the official Discord HTTP API. It also includes a lightweight HTTP server to browse the captured footage from your local browser. I built this project after a package was stolen from outside my apartment. My plan is to mount a webcam behind the peephole of my front door and connect it to a spare computer (which will run this app).

Technologies Used:

  • Language: Python 3
  • Camera Input: cv2.VideoCapture() from OpenCV
  • Video Encoding: OpenCV cv2.VideoWriter
  • Discord Integration: Python requests library using Discord Bot Token & Channel API
  • Web Server: Python http.server module via HTTPServer and BaseHTTPRequestHandler
  • Frontend: HTML, JavaScript (fetch API), JSON
  • Filesystem: Python os, datetime, and json modules for file naming, lookup, and timestamp formatting
